## 背景

`魔方助手` 最开始只是本人在学习机器视觉过程中练手的一个问题。随着后续横向纵向了解到了软件工程，发觉做好一个移动端应用远超个人的开发能力。因此决定陆续开源整个项目。

这个仓库的目标是：

1. 分析产品的需求
2. 根据技术难度和工程量决定实现哪些需求
3. 设计并定义软件架构
4. 交互、UI、编码 开发
5. 各大平台全新上架
6. 接受反馈并优化升级
7. 总结全生命周期并分享
